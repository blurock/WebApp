• General Comments
    • Click the search button (after 'Search Database' to the right, to open up the search field.
    • If you want to eliminate the field line (and all lines underneath) click the X.
    • To further search an element, click the down arrow (at the right). This uses the current element as the next search and places the results at the next sub level. In this way you are following the tree of examples (see video examples)
    • If more than five results are given at a level, a 'click for more' shows the next five.
  • The following are examples of keyword searches
    • CHEMKINMechanisms
      • A list data sets with mechanisms in CHEMKIN format
    • LLNL
      • Information about the data source LLNL
      • This includes the data sets from this source ('mechanismSource')
    • LLNL#H2
      • Information about the data source LLNL#H2, this includes the species and reactions
      • The data source, 'LLNL', is separated by the specific data set of that source, 'H2', by a '#'
    • LLNL#H2#oh
      • Information about the species 'oh' in the data source 'LLNL#H2'
      • The data set, 'LLNL#H2', is separated from the species, 'oh', by a '#'
    • c2h5oh
      • The mechanism species (with a compound name with source info like 'Galway Propene C2H5OH') which uses 'c2h5oh' as a label
    • c2h6o
      • The mechanism isomers (with a compound name with source info like 'Galway Propene C2H5OH') which have 2 carbons, 6 hydrogens and 1 oxygen
      • Order of the atoms in the isomer label is important, carbons then hydrogens then oxygens (note that, for example, that hydroxyl isomer is written 'ho').
      • Some species names could be the same as the isomer name (for example, c2h4).
    • h2o2=oh+oh
      • The lists all the reactions found with 'h2o2' as reactants and two 'oh' as products (or visa-versa).
      • Note: The species labels have to be the same in the mechanism (case-insensitive).
      • Equivalent searches are:
        • 'h2o2=2oh' (the number of species can be given)
        • 'oh+oh=h2o2' (reactants and products can be reversed)
        • 'oh + oh = h2o2' (spaces can be inserted between the species names - not spaces within species)
    • ch3+h=ch4
      • Similar to the previous example, where methyl and hydrogen form methane.
      • Third body information is not given (no M or (m), etc.)
      • The ordering of species is not important ('h+ch3=ch4' is equivalent)
